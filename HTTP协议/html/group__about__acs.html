Welcome to ACS

Amazon Common Software (ACS) for Connected Devices is a consolidation of Amazon specific middleware components (SDKs) running on a set of cross platform APIs, ensuring easy portability across a large number of HW/OS platforms.

Using this Documentation

  • Use the Navigation Tree on the left to access the ACS API Reference and the Core Documents.
  • Refer to the Release Notes ver. 1.3.0-preview to become more familiar with this release of ACS.
  • Introduction to ACS provides an overview of ACS, including architecture diagrams.
  • Quick Start Guides provide details about rapid implementation of ACS on Raspberry Pi and ESP32 reference platforms.
  • DPK Porting Guides provide specific details on device porting.
  • Middleware User Guides provide specific use cases and workflows.
  • Testing Guides provide testing instructions for specific components.
