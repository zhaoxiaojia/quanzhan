◆ aceDispatcher_listSortedInsert()

ace_status_t aceDispatcher_listSortedInsert ( aceDispatcher_listHead_t *  list,
aceDispatcher_listNode_t *  node,
aceDispatcher_listCompareFunc  fn 
)

Insert element into sorted list.

This function may be used to maintain a list in ascending or descending order. Argument 1 to the comparator is the node to insert and argument two is a node on the linked list. The insertion point is the first evaluation of comparator < 0. This produces a list sorted in the ascending order.

Parameters
[in]listPointer to the list.
[in]nodePointer to the node to insert.
[in]fnGeneric comparator that evaluates whether the first node is greater, lesser or equal to the second.
Returns
ACE_STATUS_OK on success, else error code from ace_status_t.
