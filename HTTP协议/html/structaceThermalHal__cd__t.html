Detailed Description

Cooling Device Data For Linux type, it map directly to cooling devices For RTOS, hal will map those value to cool device. i.e, cpu speed is defined as different cooling state. cb_setCDState is used when there is no thermal/ cooling manager in low level such as RTOS project. Otherwise, set to NULL such as Linux or NoART.
