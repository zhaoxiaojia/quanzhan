◆ aceDispatcher_postWork_t

typedef ace_status_t(* aceDispatcher_postWork_t) (aceDispatcher_workInternal_t *wi, void *ctx)

Post work handler.

This is the function prototype for passing work to the upper layer. It is called when delay expires.

Note
The upper layer must not execute the work inside the handler and only schedule it for execution after.
Must be non-blocking.
May be used for debugging, or for retries.
Parameters
[in]qpPointer to the work.
[in]ctxOpaque user context.
Returns
ACE_STATUS_OK on success, else error code from ace_status_t.
