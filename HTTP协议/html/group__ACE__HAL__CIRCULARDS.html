◆ aceCircularDsHal_reclaim()

ace_status_t aceCircularDsHal_reclaim ( aceCircularDs_t *  storage)

Reclaim Space for an element.

Parameters
[in]storage- handle that was returned during open
Returns
ACE_STATUS_OK on success, else one of the error codes in ace_status_t
Note
This is a thread-safe API.Try to reclaim space by erasing all previously read data. This works if there is enough data to be cleaned up i.e a block for block based devices.

aceCircularDsSync_t  type

Mark the current frame in the Storage element as read/write complete.

Parameters
[in]storage- handle that was returned during open
[in]type- Read/Write Complete aceCircularDsSync_t
Returns
ACE_STATUS_OK on success, else one of the error codes in ace_status_t
Note
This is a thread-safe API.The user should call this api with type as - ACE_CIRCULARDS_READ_COMPLETE for aceCircularDsHal_GetData. ACE_CIRCULARDS_WRITE_COMPLETE for aceCircularDsHal_SetData. once they are done reading/writing a frame.
