Detailed Description

ACE Hardware and Platform Abstraction Interfaces.

The HAL APIs as part of device porting kits APIs, abstract the major device driver (e.g. LED, button, power/thermal management, storage, audio etc.),and underlying security, connectivity infrastructure. Reference implementations of these HALs are available on a few MCU/SOC based hardware platforms. The 3P device makers can adapt these components easily to their custom hardware
