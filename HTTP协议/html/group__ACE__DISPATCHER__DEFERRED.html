typedef ace_status_t(* aceDispatcher_signalPollCb_t) (void *ctx)

Signal poll request handler.

This is the function prototype for handler provided by the upper layer to signal to it when aceDispatcher_deferredWorkMgrPoll() is required. It is called when a deferred is added, removed, or expires.

Note
Must be non-blocking.
Parameters
[in]ctxOpaque user context.
Returns
ACE_STATUS_OK on success, else error code from ace_status_t. May be used for debugging, ignored otherwise.

◆ aceDispatcher_deferredWorkMgrPoll()

ace_status_t aceDispatcher_deferredWorkMgrPoll ( aceDispatcher_deferredWorkMgr_t *  state,
aceDispatcher_postWork_t  cb,
void *  ctx 
)

Periodic deferred work manager bookkeeping.

This function manages the OSAL timer by starting/stopping as needed based on the deferred at the head of the list. Every call to this function resets the OSAL timer expiration based on current time.

Note
Once system time is set, aceTime_setClockTime should be called to ensure exact calendar time.
Parameters
[in]statePointer to the deferred work manager state.
[in]cbCallback to pass work ready for execution.
[in]ctxOpaque user context passed into cb.
Returns
ACE_STATUS_OK on success, else error code from ace_status_t.
