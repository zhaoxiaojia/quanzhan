◆ aceDiagRtcHal_callback_t

typedef void(* aceDiagRtcHal_callback_t) (aceDiagRtcHal_timerStatus_t status, void *user_context)

RTC notification callback function type. The callback function is for the notification of the status change of alarm or wakeup timer. This callback function is passed to diag RTC hardware abstraction layer adapter by aceDiagRtcHal_setCallback(). It shouldn't be invoked in ISR directly.

Parameters
[in]statusRTC timer status defined in aceDiagRtcHal_status_t.
[in]user_contextUser context passed when setting the callback function. This is not used by the driver, but just passed back to the user in the callback function.

◆ aceDiagRtcHal_ioctlRequest_t

enum aceDiagRtcHal_ioctlRequest_t

RTC ioctl request types.

Note
  • ACE_DIAG_RTC_HAL_REQ_SET_ALARM: Takes input type aceDiagRtcHal_datetime_t
  • ACE_DIAG_RTC_HAL_REQ_GET_ALARM: Returns aceDiagRtcHal_datetime_t
  • ACE_DIAG_RTC_HAL_REQ_SET_WAKEUP: Takes uint32_t and the unit is msec
  • ACE_DIAG_RTC_HAL_REQ_GET_WAKEUP: Returns uint32_t and the unit is msec
  • ACE_DIAG_RTC_HAL_REQ_GET_STATUS: Returns aceDiagRtcHal_timerStatus_t
Enumerator
ACE_DIAG_RTC_HAL_REQ_SET_ALARM 

Set alarm, when Alarm need to occur

ACE_DIAG_RTC_HAL_REQ_GET_ALARM 

Get Alarm, when Alarm will occur

ACE_DIAG_RTC_HAL_REQ_SET_WAKEUP 

Set Wakeup time in milliseconds

ACE_DIAG_RTC_HAL_REQ_GET_WAKEUP 

Get Wakeup time in milliseconds

ACE_DIAG_RTC_HAL_REQ_GET_STATUS 

Get the RTC timer status value.

◆ aceDiagRtcHal_ioctl()

ace_status_t aceDiagRtcHal_ioctl ( aceDiagRtcHal_desc_t *  rtc_descriptor,
aceDiagRtcHal_ioctlRequest_t  request,
void *  buffer 
)

Retrieves RTC status and getting/setting alarm/wakeup configuration.

Note
  • If alarm is not set, request "ACE_DIAG_RTC_HAL_REQ_GET_ALARM" is issued, this function should return error codes other than ACE_STATUS_OK.
  • If wakeup is not set, request "ACE_DIAG_RTC_HAL_REQ_GET_WAKEUP" is issued, this function should return error codes other than ACE_STATUS_OK.
  • If "ACE_DIAG_RTC_HAL_REQ_SET_ALARM" or "ACE_DIAG_RTC_HAL_REQ_SET_WAKEUP" is issued and not triggered, any incoming request about these two requests should be rejected and this function should return error codes other than ACE_STATUS_OK..
Parameters
[in]rtc_descriptorThe pointer to aceDiagRtcHal_desc_t instance returned in aceDiagRtcHal_open() call.
[in]requestIoctl request.
[in,out]bufferThe pointer to the buffer which holds the value needed by the request.
Returns
ACE_STATUS_OK on success, else one of the error codes in ace_status_t on failure.
