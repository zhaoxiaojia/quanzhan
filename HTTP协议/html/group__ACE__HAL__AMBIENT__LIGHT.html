Detailed Description

HAL API for reading Ambient Light Sensor.

Ambient Light Sensor HAL APIs provide an interface to access the hardware Ambient Light Sensor to ACS middleware component such as LED manager to enable the auto brightness feature

Steps to setup and use ALS HAL interface: 1) aceAmbientLightSensorHal_open needs to be called first to have ALS HAL APIs ready for use. 2) Once the Ambient Light Sensor HAL is open, aceAmbientLightSensorHal_getLux, aceAmbientLightSensorHal_minLux and aceAmbientLightSensorHal_maxLux can be used to retrieve sensor lux values. 3) aceAmbientLightSensorHal_close needs to be called upon exit.
