Detailed Description

HAL API for low level Thermal management.

Thermal HAL APIs provide an interface between Thermal middleware and Thermal HAL for the middleware thermal service to monitor and command a high-level SW model to take additional action based on thermal state. The middleware thermal service receives thermal state, thermal sensor reading and cooling device state through these APIs. It also applies thermal policy to thermal HAL and further to platform specific thermal management framework for mitigating thermal. If such thermal management framework is not available, optional thermal handling has to be added in middleware.
