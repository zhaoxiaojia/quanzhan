Detailed Description

DeviceInfo HAL API Definitions.

Device Info HAL APIs provide the read only access of device specific and software information. These details are typically used by middleware services such as OTA and Metrics and Logging. Here is a list of examples of device info usage:

  • OS Version
  • OS Build Variant
  • Product Name
  • Hardware Name
  • Software Package Name
  • Device Serial Number, etc

For porting suggestions, guides or typical implementation details, check Device Info Porting Guide.

)

Get Device Info entry.

Get the content for the specified Device Info entry. The caller must allocate the required memory based on the size of the entry by aceDeviceInfoDsHal_getEntrySize or use Device Info Entry Length Limits

Parameters
[in]entryEnum type of specified Device Info entry.
[in,out]dataPointer to the Device Info entry content output buffer (caller allocated).
[in]data_sizeBuffer size of the content output buffer. Mimimum buffer size is content length + 1. Refer to aceDeviceInfoDsHal_getEntrySize to get the content length.
Returns
String length of the specified entry on success, or an ace_status_t error otherwise.
Return values
ACE_STATUS_DEVICE_INFO_ENTRY_NOT_SUPPORTEDEntry is not supported on this platform.
ACE_STATUS_BAD_PARAMInput paramter is invalid.
ACE_STATUS_BUFFER_OVERFLOWContent output buffer is smaller than the entry data size.
ACE_STATUS_DEVICE_INFO_INTERNAL_ERROROther internal errors.

Get Device Info entry size.

Get the size of a specified Device Info entry.

Parameters
[in]entryEnum type of specified Device Info entry.
Returns
String length of the specified entry on success, or an ace_status_t error otherwise. Length returned is the content length without NULL terminator. When allocating memory for aceDeviceInfoDsHal_getEntry data output buffer, need to use length + 1.
Return values
ACE_STATUS_DEVICE_INFO_ENTRY_NOT_SUPPORTEDEntry is not supported on this platform.
ACE_STATUS_BAD_PARAMInput paramter is invalid.
ACE_STATUS_DEVICE_INFO_INTERNAL_ERROROther internal errors.
