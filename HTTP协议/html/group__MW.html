Detailed Description

ACE Portable Middleware.

ACE is a consolidation of Amazon specific middleware components (SDKs) running on a set of cross platform APIs, ensuring easy portability across a large number of HW/OS platforms. In addition to the portability, ACE is designed to provide a highly configurable software distribution model for constituent middleware components. ACE also offers a test infrastructure to validate the different layers of the software stack.
