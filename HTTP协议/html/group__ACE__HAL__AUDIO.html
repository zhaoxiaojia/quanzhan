Detailed Description

API for Audio functionality.

Audio HAL provides the interfaces to control the audio devices. audio device state/control/open/close/read/write.

Audio HAL APIs provide primitive Audio functionalities to upper Audio manager to control low level Audio devices. It handles audio device state, control, and audio IO data read, write and control.

The normal working steps as following:

  1. aceAudDev_getDevice
  2. aceAudDev_open
  3. aceAudDev_write/aceAudDev_read
  4. aceAudDev_close
