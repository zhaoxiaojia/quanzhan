Detailed Description

Diagnostic HAL APIs for storage accesses.

◆ aceDiagStorageHal_desc_t

typedef struct aceDiagStorageHal_desc aceDiagStorageHal_desc_t

aceDiagStorageHal_desc_t is the type of storage descriptor structure returned by calling aceDiagStorageHal_open(). It is initialized in aceDiagStorageHal_open() and returned to the caller.

Enumeration Type Documentation

◆ aceDiagStorageHal_operation_t

enum aceDiagStorageHal_operation_t

the operation mode for diag storage IO control.

Enumerator
ACE_DIAG_STORAGE_HAL_OP_RDONLY 

Open storage with read only mode

ACE_DIAG_STORAGE_HAL_OP_WRONLY 

Open storage with write only mode

ACE_DIAG_STORAGE_HAL_OP_RDWR 

Open storage with read/write mode
