Detailed Description

APIs for Audio Manager functionality.

Audio Manager HAL provides the APIs to control the audio manager.

The Audio Manager HAL APIs expose primitive audio-related functionality to the middleware control-layer, allowing it to talk with the lower-level audio service on the target.
