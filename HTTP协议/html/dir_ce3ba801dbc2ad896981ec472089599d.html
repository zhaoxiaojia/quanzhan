file  hal_diag_adc.h
 Diag ADC HAL is a diagnostic ADC (Analog-to-Digital Converter) hardware abstraction layer for the diagnostic test service, and these APIs are very test oriented. If there is more than one task or process other than the diagnostic test service accessing the same ADC resource, the developer should handle race conditions and resource control under other layers.
