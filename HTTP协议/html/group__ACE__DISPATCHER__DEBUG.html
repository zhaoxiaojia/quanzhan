◆ aceDispatcher_debugInit()

ace_status_t aceDispatcher_debugInit ( void  )

Initialize the debug module.

This function must be called to initialize the debug module before any other API in this header is used.

Note
Caller may ignore return values as errors in debug module does not impair core functionality.
Warning
Must not be called more than once. It is assumed to be called once at boot up and never again until next reboot.
Returns
ACE_STATUS_OK on success, else error code from ace_status_t.
