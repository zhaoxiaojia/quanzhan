Detailed Description

HAL API for Virtual Key Input.

Virtual Key HAL APIs are set of APIs provided and implemented by the platform to pass the virtual key event to the middleware component such as virtual key input manager. The virtual key event is typically used by software key event, or Bluetooth AVRCP and HID profile events.

Steps to setup and use Virtual Key HAL interface:
1) aceVirtualKeyHal_open() needs to be called first to set up the detection thread ready for use.
2) Set up the timeout value, callback function, and enablement via aceVirtualKeyHal_setParams(). aceVirtualKeyHal_getParams() can be used to retrieve current activation state.
3) Use aceVirtualKeyHal_close() to exit.
