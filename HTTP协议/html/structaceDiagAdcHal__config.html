Detailed Description

data structure for ADC ioctl request ACE_DIAG_ADC_HAL_REQ_GET_CONFIG where both sample_time_cycles and resolution_bits are output results.

Note
  • sample_time_cycles: Sample time in ADC clock cycles, applies to all channels. Shortening sample time maximizes conversion speed for lower impedance input. Extending sample time improves sample accuracy for higher impedance input.
  • resolution_bits: ADC channel resolution, 12 = 12bits, 13 = 13bits, 16 = 16bits, 24 = 24bits etc
