)
private

Parse the custom config file.

This function is to be use by the Linux build systems only. It reads the config line by line from a config file denoted by the file pointer fp and constructs the ace_config_t out of it. The config file will be in TOML format.

Returns
ACE_STATUS_OK on success, else error code ACE_STATUS_NULL_POINTER.
