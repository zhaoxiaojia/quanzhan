◆ aceWifiMgr_networkState_t

enum aceWifiMgr_networkState_t

Network State Describes the current state of the network. When registered for network state change events, the state will only be published at the moment of the state change. e.g. NET_STATE_DISCONNECTED will only be published if the device is already connected. Similarly, NET_STATE_CONNECTED will only be published if the device was not already in the connected state.

Enumerator
NET_STATE_CONNECTING 

The device is currently connecting

NET_STATE_CONNECTED 

The device is connected

NET_STATE_DISCONNECTING 

The device is disconnecting

NET_STATE_DISCONNECTED 

The device is disconnected

NET_STATE_UNKNOWN 

The network state is unknown
