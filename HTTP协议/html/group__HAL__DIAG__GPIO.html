◆ aceDiagGpioHal_callback_t

typedef void(* aceDiagGpioHal_callback_t) (aceDiagGpioHal_value_t pin_state, void *user_context)

GPIO interrupt callback type. This callback is passed to diag gpio hardware abstraction layer adapter by using aceDiagGpioHal_setInterruptCallback() API. This callback shouldn't be passed to ISR context.

Parameters
[in]pin_stateThe vaule of the GPIO pin in aceDiagGpioHal_value_t.
[in]user_contextUser Context passed when setting the callback. This user contenxt can't be used in ISR, but just passed back to the user in the callback.

◆ aceDiagGpioHal_desc_t

typedef struct aceDiagGpioHal_desc aceDiagGpioHal_desc_t

aceDiagGpioHal_desc_t is the type of GPIO descriptor structure returned by calling aceDiagGpioHal_open(). It is initialized in aceDiagGpioHal_open() and returned to the caller.

Creates an instance for desired GPIO relevant operations.

Parameters
