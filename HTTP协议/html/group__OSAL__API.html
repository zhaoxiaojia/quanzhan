Detailed Description

OS Abstraction Layer Interfaces.

OSAL provides a layer of operating system abstraction for ACE modules to run on. ACE provides a default implementation of OSAL through POSIX complaint libs (e.g. GLIBC) for POSIX complaint operating systems, or on FreeRTOS for real-time operating systems. Other ports may also be available. If using one of the supported OSes, the device can directly use the OSAL ports on their hardware platform. Otherwise they need to implement the OSAL APIs.
