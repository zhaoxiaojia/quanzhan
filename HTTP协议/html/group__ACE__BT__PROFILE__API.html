◆ aceBT_a2dpSink_connectProfile()

aceBT_status_t aceBT_a2dpSink_connectProfile ( aceBT_bdAddr_t *  p_remoteDevice)

API to initiate a Profile connection procedure for the remote device. This is an asynchronous call and the callbacks for connection states will be sent as part of profile_state_callback().

Parameters
[in]p_remoteDevicepointer to remote device address aceBT_bdAddr_t
Returns
ACEBT_STATUS_SUCCESS if success
ACEBT_STATUS_UNHANDLED if unsupported profile
ACEBT_STATUS_NOMEM if ran out of memory
ACEBT_STATUS_BUSY if profile is busy connecting another device
ACEBT_STATUS_PARM_INVALID if request contains invalid parameters
ACEBT_STATUS_NOT_READY if server is not ready
ACEBT_STATUS_FAIL for all other errors

API implementation to initiate a A2DP Sink profile connection procedure for the remote device.
