◆ aceBT_bleConnPriority_t

enum aceBT_bleConnPriority_t

BLE connection priority set as part of Connection parameter update.

Enumerator
ACEBT_BLE_CONN_PRIORITY_CRITICAL 

Priority critical : High priority, ultra-low latency param. This should be used to transfer large amount of data quickly. Applications should switch to BALANCED or LOW priority once done.

ACEBT_BLE_CONN_PRIORITY_HIGH 

Priority high : high priority, low latency param

ACEBT_BLE_CONN_PRIORITY_BALANCED 

Priority balanced : Uses default conn param recommended by Bluetooth SIG

ACEBT_BLE_CONN_PRIORITY_LOW 

Priority low : Low power, high latency param
