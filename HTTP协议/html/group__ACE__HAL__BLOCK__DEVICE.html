Detailed Description

HAL API for block device I/O.

Block Device HAL APIs provide the basic read, write and erase I/O operations on a block-based storage. A block device is a bounded continuous byte addressable block-based I/O storage. It can be either a physical device or a virtual device.

block device: a bounded continuous byte addressable block-based I/O storage
