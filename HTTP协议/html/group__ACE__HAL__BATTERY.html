Detailed Description

HAL API for low level battery management.

Battery HAL APIs provide an interface between Battery MW and Battery HAL for the Battery MW service to monitor battery state and health. The Battery MW service receives battery level, status, charging state, source, voltage, current, battery temperature, etc. through these APIs. For RTOS, if such battery management framework is not available, optional battery management/ monitoring has to be added in MW.
